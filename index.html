Oblivio

Oblivio is a simple Docker-based app to run Ollama local LLMs with an easy chat UI and quick model switching. To use it, Ollama must be installed beforehand. It makes local AI hassle-free for everyone.

Install with Docker:

 docker pull rgulersen/oblivio:latest

run:

 docker run -p 5000:5000 rgulersen/oblivio
